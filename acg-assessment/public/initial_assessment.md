# ACG Candidate Technical Assessment

**Please complete the following tasks on your own computer and submit your results at the bottom of this screen.**

This assessment is designed to evaluate how you approach tasks in a remote work environment, including attention to detail, ability to follow instructions, communication clarity, and problem-solving.

***

## 1. Internet Connectivity Check

- Connect to your home WiFi.  
- Run a speed test (any provider is acceptable).  
- Take a screenshot of the results.  

If your speed test is below **25 Mbps download** or **5 Mbps upload**:
- Include one sentence in your final email describing what steps you would take to improve it.

***

## 2. Initial Email Submission

Send an email to the provided test address with:

- A professional subject line.  
- A professional greeting.  
- A concise message (3–5 sentences) explaining why you are a good fit.  
- Attach your speed test screenshot.

**File Naming Requirement:**
Rename the file as: `YourLastName-WiFiTest.png`

***

## 3. Browser and File Management Task

- Open at least **3 tabs** in your web browser.  
- Take a screenshot of your screen showing those tabs.

Then:
- Create a folder on your Desktop named: `ApplicationScreenshots`
- Save your screenshot in that folder using the exact file name: `2026-04-15_[YourLastName]_BrowserScreenshot.png`  
  *(Use today’s date in YYYY-MM-DD format. Replace `[YourLastName]` with your actual last name. No spaces.)*

***

## 4. Documentation Task

Inside the same folder, create a text file named: `Notes_[YourLastName].txt`

In this file, write 2–3 sentences summarizing:
- The steps you completed.  
- Any issues you encountered (if applicable).

***

## 5. Final Email Reply

Reply to the original test email and:
- Attach the correctly named browser screenshot.  
- Confirm the file is stored in the correct folder.  
- Include your **start time and completion time**.

***

## 6. Problem-Solving Check

One or more steps in this exercise may not work exactly the same on every system.

If you encounter any issues:
- Briefly explain what happened in the text box below.  
- Describe how you resolved it.
